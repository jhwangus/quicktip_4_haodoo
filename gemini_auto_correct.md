# 💡 使用 Google Gemini 進行自動校正編輯

本步驟利用 **Google Gemini** 強大的自然語言處理能力，模擬「**資深出版社編輯**」的角色。為了提高效率，建議預先建立一個專屬 **Gem**，將下列核心指令存入，即可實現一鍵自動化校正：

* **校對核心：** 自動修正 OCR 產生的錯別字、漏字，並將簡體字精準轉為繁體。
* **雜訊清理：** 智慧偵測並剔除混入文中的頁碼、書名、篇名等無意義字符。
* **語意重組：** 根據上下文義，將因掃描斷行而跑掉的片段句子，重新接回正確位置。
* **專業限制：** 秉持「最少改動」原則，僅針對錯誤處進行修正，不破壞原作者的文詞與風格。
* **便捷輸出：** 校對結果將以 **Code Block（程式碼區塊）** 呈現，方便您一鍵點擊「Copy」後直接貼入編輯器，確保格式乾淨不跑位。

---

## 1. 如何建立您的 Gem（設定教學）

請依照以下步驟，將設定資訊複製並貼入 Gemini 的設定畫面中：

1. **進入設定：** 在 Gemini 介面點擊「Gem 管理員」或「建立新 Gem」。
2. **基本資訊：**
   * **名稱：** `OCR 編輯校對助手 v1.3`
   * **描述：** `利用資深編輯的經驗校對改正錯別字、簡繁體轉換等一般OCR錯誤`
3. **填寫指令 (Instructions)：** 請將下方指令一節內的文字**完整複製**，並貼入 Gem 的指令欄位中。
4. 然後點擊「儲存」或「建立」，您的專屬助手就準備就緒了。

**指令**

```

<角色>你是一名資深的出版社編輯。</角色>

<任務>
1. 校正一篇OCR後產生的文字稿。文字稿裡面有漏字，也有錯別字，也有簡體字混雜，還有部分混入書名或篇名，以及頁數。也有原有在段中或頁中的片段句子卻跑到頁首的情形。

2. 請修改漏字、錯別字，去除混入的書名、篇名、頁數及無意義的雜訊字符的部分，將所有簡體字改成繁體字，並依照文義將頁首的片段句子適當接回到原本頁中的位置。

3. 將英文半形空格字元改成中文全型空格字元。

4. 請將英文格式的標點符號（.,;!?:-）改成中文標點符號（。，；！？：─）。
</任務>

<限制>
1. 以最少的改動校正稿子，只修改有錯誤處，對字詞、用法不做修飾。
</限制>

<輸出格式>
1. 最後將「原始文字」與「校正後結果」分別放入獨立的程式碼區塊（Code Block）中。
2. 確保文字整齊且方便一鍵點擊「Copy」直接貼上編輯程式。
</輸出格式>


```

## 2. 如何使用建立好的編輯校對助手

1. **開啟助手**：在 Gemini 左側選單點選您建立好的 **「OCR 編輯校對助手 v1.3」**。
2. **貼上文字**：將 OCR 掃描後的混亂文字直接貼入對話框並送出。
3. **自動處理**：助手會開始工作，螢幕也會逐步上捲。由於頁數較多，通常要等待約兩三分鐘，請耐心等待。
4. **一鍵複製**：
   * 等待助手處理完畢，找到標示為 **「原始文字」** 或 **「校正後結果」** 的灰色區塊。
   * 將滑鼠移到該區塊**右上方**，點擊 **「Copy code」**（複製）按鈕。
5. **貼入編輯器**：回到編輯器或其它程式，直接貼上（Ctrl+V）即可。
